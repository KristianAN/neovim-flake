# Neovim dotfiles

Minimal neovim configuration

All configuration is predominantly written in fennel. Where runtime information is required the configuration is written in nix with lua

# Configured language support
  - Haskell
  - Rust
  - Scala
  - Java
  - Clojure
  - Fennel
  - Lua
  - TypeScript
  - C#
  - F#
  - Python
